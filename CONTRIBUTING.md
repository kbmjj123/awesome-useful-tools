The following is a set of guidelines for contributing to this repository.

## How to contribute

1. **Fork the repository**: Click the "Fork" button at the top right of this page.
2. **Clone your fork**: Clone the repo to your local machine (or edit directly on GitHub).
3. **Create a branch**: `git checkout -b add-new-tool`
4. **Add your content**:
    - Please ensure the tool is actually useful and reliable.
    - Check if the tool is already listed.
    - Add the link to the appropriate category.
    - **Format**: `[Tool Name](URL) - Short description.`
5. **Commit your changes**: `git commit -m "Add [Tool Name]"`
6. **Push to the branch**: `git push origin add-new-tool`
7. **Submit a Pull Request**: Open a PR to the `main` branch of this repository.

## Guidelines

- Keep descriptions short and simple.
- Do not use affiliate links.
- Ensure the link is working.
- Respect the alphabetical order if possible.

Thank you for your help!
